# 1. 개요 
### 프로젝트 이름 : Shopping
### 개발 기간 : 2023년 11월 ~
### 스택 
  - HTML, CSS, JavaScript, React
  - 상태 관리 : Redux
  - 데이터 관리 : FireBase
### 기능
  - 제품 관리
    + 카테고리를 통한 제품 분류
  
  - 장바구니 기능
    + 사용자가 상품을 장바구니에 담을 수 있음
    + 장바구니에서 수량 조절

 - 주문 및 결제
   + 주문하기 기능
   + 결제하기 기능

- 사용자 계정
  + 회원가입 및 로그인
  + 주문 내역과 배송 상태 확인

- 검색 기능
   + 브랜드, 상품 이름을 통한 제품 필터링
